## Features

- **Multiple Factory Patterns**: Implementation examples of various factory design patterns
- **Abstract Product Factory**: Demonstrates factory pattern for abstract base classes
- **Widget Factory**: Concrete implementation of factory pattern for widget creation
- **Interface Programming**: Examples of programming to interfaces rather than implementations
- **Educational Examples**: Well-structured code examples for learning factory patterns
- **Cross-Platform Compatibility**: C++ implementations that work across different platforms

## Installation

### Prerequisites

- C++ compiler with C++11 support or later (GCC, Clang, or MSVC)
- Visual Studio (recommended for Windows development)
- Git for cloning the repository

```bash
git clone https://github.com/JimFawcett/ObjectFactories.git
```

## Usage

### Basic Factory Pattern Example

```cpp
#include "Factory.h"

int main() {
    // Create factory instance
    auto factory = std::make_unique<ConcreteFactory>();
    
    // Create objects using factory
    auto product = factory->createProduct("ProductType");
    
    // Use the created object
    product->operation();
    
    return 0;
}
```

### Abstract Factory Pattern Example

```cpp
#include "AbstractFactory.h"

int main() {
    // Get factory for specific product family
    std::unique_ptr<AbstractFactory> factory = 
        FactoryProvider::getFactory("ConcreteFamily");
    
    // Create related products
    auto productA = factory->createProductA();
    auto productB = factory->createProductB();
    
    // Use products together
    productA->interact(productB.get());
    
    return 0;
}
```
